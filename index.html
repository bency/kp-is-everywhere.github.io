柯文哲關鍵字小幫手

政見太多沒時間看，不知道哪些政見跟你有關嗎？
馬上安裝 Chrome 擴充套件 柯文哲關鍵字小幫手，讓你在瀏覽網頁時，輕鬆得到相關資訊！

簡介

目前柯文哲的政見已超過 15 條，想要一篇一篇看是相當花時間的。
安裝了這個之後，它就會陪你一起看網頁，當關鍵字出現，就會介紹一條相關的政見，也許就剛好會看到你有興趣的政見囉！

誠心推薦您－偷偷貼心地裝在爸媽的電腦上。

jQuery Plugin

想放在自己的網站上嗎？ 柯文哲關鍵字小幫手也準備好了 jQuery Plugin，
讓你可以巧妙地放在你的個人網站喔！（明日台北之類的也是相當適用）

使用說明

  1. 下載

    kpkey.js 目前相依於 jquery.js, tabletop.js 和 highlight.js，所以請下載這個壓縮檔

  2. 加入 css 到頁面

    放在 <head></head> 裡 

    <head>
  <link href="css/kp-highlight.css" media="screen" rel="stylesheet" type="text/css" />
</head>

  3. 加入 js 到頁面

    把它們和 jquery 加到的 </body> 之後 

    <script src="js/jquery-2.1.1.min.js" type="text/javascript"></script>
<script src="js/tabletop.js" type="text/javascript"></script>
<script src="js/highlight.js" type="text/javascript"></script>
<script src="js/kpkey.js" type="text/javascript"></script>
<script type="text/javascript">
    $('.example').kpkey();
    // .example 是要搜尋的元素，也可以用 $('body') 做全頁搜尋
</script>

  4. 也不能忘記柯 P 的照片（意思就是你也可以換成你喜歡的啦）

    把 kp.jpg 放到和 js 同層的 images 資料夾裡

  5. have fun!
